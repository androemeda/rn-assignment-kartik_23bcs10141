# Setup Instructions

## Install dependencies

```bash
npm install
```

## Start the app

```bash
npm start
```

## Features

✅ Length conversion (Meters ↔ Feet)  
✅ Temperature conversion (Celsius ↔ Fahrenheit)  
✅ Live conversion as you type  
✅ Input validation (numbers only)  
✅ Swap units button  
✅ Clear input button  
✅ TypeScript implementation  
✅ Tailwind CSS styling
